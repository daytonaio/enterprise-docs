Daytona allows you to connect to your Workspace using your default IDE.

Out of the box, Daytona can be configured to open your Workspace in the following IDEs:

- **Visual Studio Code (VSCode)**

  - [Desktop (Linux, macOS, Windows)](#vs-code)
  - [Browser](#vs-code-browser)

- **JetBrains IDEs**
  - [CLion](#jetbrains-clion)
  - [GoLand](#jetbrains-goland)
  - [IntelliJ IDEA Ultimate](#jetbrains-intellij-idea-ultimate)
  - [PhpStorm](#jetbrains-phpstorm)
  - [PyCharm Professional](#jetbrains-pycharm-professional)
  - [Rider](#jetbrains-rider)
  - [RubyMine](#jetbrains-rubymine)
  - [WebStorm](#jetbrains-webstorm)

- [**Cursor**](#cursor)

## Set the Default IDE

You can set the default IDE used by Daytona. Daytona allows you to connect to your development environment using a variety of IDEs, including popular options like Visual Studio Code (VSCode) and JetBrains IDEs.

By setting the default IDE, you can automatically open your development environment in your preferred environment, optimizing your workflow and reducing setup time.

1. In the Daytona dashboard, navigate to your profile account icon and select **`Settings`**.
2. Click **`Default IDE`** dropdown menu to select your default IDE.

## Set the Workspace IDE

You can set the default Workspace IDE used by Daytona. Daytona allows you to connect to your development environment using a variety of IDEs, including popular options like Visual Studio Code (VSCode) and JetBrains IDEs.

By setting the default Workspace IDE, you can automatically open your development environment in your preferred environment, optimizing your workflow and reducing setup time.

1. Expand the selected Workspace to view its configuration and details.
2. Click the **`Launch remote environment`** dropdown menu to select your default IDE for the Workspace.

Upon selecting, Daytona will set it as the default IDE for the selected Workspace. You can now automatically open the Workspace with the default IDE already connected.

## VS Code

Daytona allows you to connect to your Workspace using Visual Studio Code (VSCode) IDE.

1. In the Daytona dashboard, navigate to your profile account icon and select **`Settings`**.
2. Click **`Default IDE`** dropdown menu and select **`VS Code`** as your default IDE.

Upon selecting, Daytona will set VS Code as the default IDE for opening Workspaces. You can now automatically open the Workspace with the VS Code IDE.

Opening a Workspace with the Cursor IDE will prompt you to open it as an external application. If in full screen, navigate to your desktop to view the prompt. Click **`Open`** to proceed with opening the Workspace in the Cursor IDE. Installing the *Remote - SSH* extension within the IDE is required to open the remote window.
:::
